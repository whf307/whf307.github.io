This script is used to print and edit block offset of PostgreSQL pages,in python 2.7 .And you can use it only on Linux,if you want to use it on other platfrom like AIX ,maybe you could edit it again youself,good luck! Just find the $PGDATA,oid of database and relfilenode into ./file and then u can use it now! For $PGDATA excute "echo $PGDATA" or "ps -ef | grep postgres" to find the data directory!!! For others,just run "select pg_relation_filepath('t1');" to find the file path. And edit the ./file like '/oracle/soft/pg_data/base/13323/16422' And then ,run chmod 755 bbedp,and execute "./bbedp" ,the new world is here. If you have any problem about it, please concat with me.
